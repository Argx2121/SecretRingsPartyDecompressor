# SecretRingsPartyDecompressor
Python script to decompress Sonic and the Secret Rings Party mode files\
Works with python 3.11 and probably other versions
## Usage
Move the python file in your folder with the compressed files and run it via cmdline (eg. python SecretRingsPartyDecompressor.py)\
